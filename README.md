## 🚀 How to Use

1. Clone or download this repository
2. Open `index.html` in a web browser
3. Or deploy to a web host (GitHub Pages, Netlify, etc.)

## 💻 Deploying on GitHub Pages

1. Go to your repository settings
2. Scroll to "Pages"
3. Select "Deploy from a branch"
4. Choose the `main` branch
5. Your site will be live at: `https://wcl5lex.github.io/lex-electric-website`

## 📝 Customization

To update the website:
1. Edit `index.html` to change content
2. Modify `styles.css` to adjust colors/design
3. Commit changes and push to GitHub
